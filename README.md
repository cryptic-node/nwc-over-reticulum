# nwc-over-reticulum
Send lightning payments and zaps over reticulum

let someone authorize a Lightning payment from an off-grid device, with the NWC request traveling over LoRa/Reticulum until it reaches a node with internet, then hits the Lightning wallet over Nostr relays. BTC Mesh exists for on-chain, but nobody has done NWC — which is the interesting half because NWC is interactive, budget-constrained, and works with any modern wallet.
